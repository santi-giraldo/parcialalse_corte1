# parcialalse_corte1

Funcionamiento general:

Solicita al usuario la cantidad de valores (n) que desea promediar

Valida que n sea mayor que cero

Si no lo es, muestra un mensaje de error y termina el programa

Declara un arreglo valores de tamaño n

Pide al usuario que ingrese cada uno de los valores

Llama a la función calcularPromedio, que:

Recorre el arreglo

Suma todos los elementos

Divide la suma entre n

Muestra el promedio calculado en pantalla
